[prerequisite]
you need to install the following tools inorder to build/run the demo:

1.npm
find more info on http://nodejs.org/

2.nodejs
find more info on https://npmjs.org/

3.grunt
find more info on http://gruntjs.com/

[how-to-run]
type the following command in command line(appy for both unix and non-unix system):

1.install required package dependency
cd ${your-cwd}/qing/demo;
npm install;
grunt server;

2.start demo 
node app.js

3.open brower for "localhost:3000" to check out how it looks
